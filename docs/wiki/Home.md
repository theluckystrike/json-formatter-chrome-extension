# JSON Formatter -- Open Source Chrome Extension

**Format, validate, and beautify JSON directly in your browser.** JSON Formatter is a free, open source Chrome extension built for developers who work with JSON every day.

**[Install from Chrome Web Store](https://chromewebstore.google.com/detail/json-formatter-pro/gbnadjkeegkhbcoeaeaoedpojlcknnhp)** | **[View Source on GitHub](https://github.com/theluckystrike/json-formatter-chrome-extension)**

---

## What It Does

JSON Formatter provides a fast, private, and feature-rich JSON viewer right inside Chrome. Paste JSON into the popup or navigate to any JSON endpoint, and the extension will format it with syntax highlighting, a collapsible tree view, and real-time validation. Everything runs locally in your browser -- your data never leaves your machine.

## Key Features

- **Format and beautify** JSON with configurable indentation (2, 4, or 8 spaces)
- **Minify** JSON to compact single-line output
- **Validate** with detailed error messages including line and column numbers
- **Collapsible tree view** for exploring nested structures with search
- **Auto-detect JSON** on web pages with a floating "Format" button
- **Syntax highlighting** with dark and light themes
- **Copy** formatted JSON to clipboard or **download** as a file
- **Right-click context menu** to format, validate, or minify selected text
- **Sort keys** alphabetically (optional)
- **Keyboard shortcuts** -- Ctrl+Shift+F to format, Tab to insert spaces
- **JSON statistics** -- real-time key count, nesting depth, and file size
- **Multi-language support** -- English, Spanish, French, German, Japanese, Chinese, Portuguese

## Privacy and Security

JSON Formatter is 100% open source under the MIT license. It collects no data, includes no analytics or tracking, makes no network requests, and runs entirely in your browser. It is built on Chrome's Manifest V3 platform with a strict Content Security Policy.
