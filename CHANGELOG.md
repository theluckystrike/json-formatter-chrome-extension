# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-01

### Added
- Format and beautify JSON with configurable indentation (2, 4, or 8 spaces)
- Minify JSON to compact single-line output
- Validate JSON with detailed error messages (line and column numbers)
- Collapsible tree view for exploring nested JSON structures
- Search within tree view by key or value
- Copy formatted JSON to clipboard
- Download formatted JSON as a file
- Auto-detect and format JSON on web pages (content script)
- Right-click context menu: format, validate, and minify selected JSON
- Auto-paste from clipboard when opening the popup
- Dark and light theme support
- Sort keys alphabetically (optional)
- Line numbers with error highlighting
- JSON statistics in status bar (key count, depth, file size)
- Keyboard shortcut: Ctrl+Shift+F to format
- Tab key inserts spaces in the editor
- Multi-language support (English, Spanish, French, German, Japanese, Chinese, Portuguese)
- Onboarding flow for new users
- Manifest V3 for maximum security and performance
